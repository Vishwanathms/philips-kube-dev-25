
✅ Create NGINX as a **Pod**
✅ Expose it using **NodePort**
✅ Access it **from outside Minikube**

---

## ✅ 1. Create NGINX Pod (direct CLI)

```bash
kubectl run nginx-pod --image=nginx --port=80 --restart=Never
```

Check pod:

```bash
kubectl get pods -o wide
```

---

## ✅ 2. Expose Pod using NodePort

```bash
kubectl expose pod nginx-pod --type=NodePort --port=80
```

Verify service:

```bash
kubectl get svc nginx-pod
```

---

## ✅ 3. Get Minikube IP

```bash
minikube ip
```

---

## ✅ 4. Get NodePort

```bash
kubectl get svc nginx-pod -o jsonpath='{.spec.ports[0].nodePort}'
```

---

## ✅ 5. Access NGINX from Outside (Browser / Curl)

Replace values from above output:

```bash
curl http://<MINIKUBE-IP>:<NODE-PORT>
```

Example:

```bash
curl http://192.168.49.2:31234
```

---

## ✅ 6. Alternative One-Command Access

```bash
minikube service nginx-pod
```

---

## ✅ 7. Cleanup (Optional)

```bash
kubectl delete pod nginx-pod
kubectl delete svc nginx-pod
```
